How to Install Tor Browser in Ubuntu

by Mohamed Shahat

Introduction

Tor Browser allows you to browse the internet anonymously, ensuring your privacy and security. This guide will walk you through installing Tor Browser on Ubuntu.

Step 1: Update Your System

First, ensure your system is up to date. Open a terminal and run the following commands:

sudo apt update && sudo apt upgrade

Step 2: Add Tor Repository

Next, you need to add the official Tor repository to your system by running the following:

sudo add-apt-repository ppa:torproject/torbrowser

Step 3: Install Tor Browser

After adding the repository, you can install Tor Browser:

sudo apt install torbrowser-launcher

Step 4: Launch Tor Browser

Once the installation is complete, you can launch Tor Browser from your application menu, or run the following command:

torbrowser-launcher
